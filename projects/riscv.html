RISC-V CPU Design

In Progress

Overview

Currently designing and implementing a 32-bit RISC-V CPU from scratch as part of UC Berkeley's CS61C (Computer Architecture) course. This project involves building a fully functional processor capable of executing RISC-V assembly instructions using hardware description in Logisim.

Planned Features

  • Two-Stage Pipeline: Will implement instruction fetch and execute stages with hazard detection
  • ALU Design: Building arithmetic logic unit to support all RISC-V R-type and I-type instructions
  • Register File: Designing 32-register file with simultaneous read/write capabilities
  • Control Logic: Developing control unit for instruction decoding and datapath coordination
  • Memory Interface: Integrating instruction and data memory with proper addressing

Technologies & Tools

Logisim RISC-V ISA Digital Logic Hardware Design Assembly Language

What I'm Learning

This project is providing hands-on experience with low-level computer architecture concepts, including instruction set architectures, datapath design, control logic, and the relationship between hardware and software. It's deepening my understanding of how high-level code translates to machine instructions and executes on physical hardware.
